## Prerequisites

Before installing mod-tools CLI, ensure you have the following:

- Access to the cslol-manager source repository
- Rust toolchain (if building from source)
- League of Legends installation (for game-specific operations)

## Building from Source

The mod-tools CLI is part of the cslol-manager project and needs to be built from source:

```bash
# Clone the repository
git clone https://github.com/LeagueToolkit/cslol-manager.git
cd cslol-manager

# Build the project
cargo build --release
```

The compiled binary will be located at `target/release/mod-tools`.

## Installation

After building, you can install the CLI globally:

### Linux/macOS

```bash
# Copy to a directory in your PATH
sudo cp target/release/mod-tools /usr/local/bin/

# Make it executable
sudo chmod +x /usr/local/bin/mod-tools
```

### Windows

```powershell
# Copy to a directory in your PATH
copy target\release\mod-tools.exe C:\Windows\System32\
```

Or add the `target/release` directory to your system PATH.

## Verifying Installation

Verify the installation by running:

```bash
mod-tools --help
```

You should see a list of available commands and options.
